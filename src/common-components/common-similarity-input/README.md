## common-similarity-input
### 组件说明
`common-similarity-input`全局相似度组件。

### 基本用法
```
// 使用
<common-similarity-input v-model="value">
</common-similarity-input>
```

### 参数说明
| 参数 | 说明 | 类型 | 可选值 | 默认值 | 备注 |
| :-----------: | :-----------: | :------: | :---------: | :--------: | :---------------------: |
| v-model | 当前绑定数值 | Number | Number | 1 | - |
